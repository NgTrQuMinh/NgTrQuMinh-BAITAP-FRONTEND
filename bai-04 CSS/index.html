Câu 1: Bài tập về selecter (bộ chọn)

Tạo thẻ p như này và thêm class="color-red". Sau đó viết thêm css để dòng này thành màu đỏ (Dùng bộ chọn class - class selecter).

Tạo thẻ p như này và thêm class="color-grren". Sau đó viết thêm css để dòng này thành xanh lá (Dùng bộ chọn id - id selecter).

Câu 2: Bài tập về background

Yêu cầu 1: Tạo 1 thẻ div, đặt tên class cho thẻ div này tùy ý, sau đó viết css theo tên class đã đặt. Chiều rộng 200px, chiều cao 100px. Có background màu xanh lá

Yêu cầu 2: Tạo 1 thẻ div, đặt tên class cho thẻ div này tùy ý, sau đó viết css theo tên class đã đặt. Chiều rộng 100%, chiều cao 200px. Có background hình ảnh (ảnh tùy ý), khi scroll hình ảnh đứng yên.
